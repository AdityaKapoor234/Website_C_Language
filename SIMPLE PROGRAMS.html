Q9) EXPLAIN THE WORKING OF SHORTHAND ASSIGNMENT OPERATORS, PRE AND POST INCREMENT OPERATOR AND THE TERNARY OPERATOR
ANS)

SHORTHAND ASSIGNMENT OPERATOR

Shorthand Assignment Operators are used to express the syntax that are already available in shorter way.

NAME OF THE OPERATOR EQUIVALENT CONSTRUCT OPERATOR SYMBOL EXAMPLE
Addition x=x+2 += x+=2
Subtractions x=x-2 -= x-=2
Multiplication x=x*2 *= x*=2
Division x=x/2 /= x/=2
Remainder x=x%2 %= x%=2


ADDITION ASSIGNMENT

It is used to add the value of a particular variable(Like x) by a specific value(Like 2)

Syntax-: x+=2

SUBTRACTION ASSIGNMENT

It is used to subtract the value of a0 particular variable(Like x) by a specific value(Like 2)

Syntax-: x-=2

MULTIPLICATION ASSIGNMENT

It is used to multiply the value of a particular variable(Like x) by a specific value(Like 2)

Syntax-: x*=2

DIVISION ASSIGNMENT

It is used to divide the value of a particular variable(Like x) by a specific value(Like 2)

Syntax-: x/=2

REMAINDER ASSIGNMENT

It is used to find the remainderof the value of a particular variable(Like x) by a specific value(Like 2)

Syntax-: x%=2

EXAMPLE-: 
#include < iostream >
#include < conio.h >
using namespace std;
main()
{
int x=10,x2=10,x3=10,x4=10,x5=10;
cout<<"SHORTHAND ASSIGNMENT OPERATORS\n\nADDITION ASSIGNMENT [x=10] (x+=2)\n";
x+=2;
cout<<"Ans) "<< x;

cout<<"\n\nSUBTRACTION ASSIGNMENT [x2=10] (x2-=2)\n";
x2-=2;
cout<<"Ans) "<< x2;

cout<<"\n\nMULTIPLICATION ASSIGNMENT [x3=10] (x3*=2)\n";
x3*=2;
cout<<"Ans) "<< x3;

cout<<"\n\nDIVISION ASSIGNMENT [x4=10] (x4/=2)\n";
x4/=2;
cout<<"Ans) "<< x4;

cout<<"\n\nREMAINDER ASSIGNMENT [x5=10] (x5%=2)\n";
x5%=2;
cout<<"Ans) "<< x5;
getch();
}


PRE INCREMENT

IT IS REPRESENTED BY (++) BEFORE THE VARIABLE NAME.

Syntax-: ++a

It increment the value of operand by 1. This operator first increments the value of operand and then returns the incremented value.

EXAMPLE-: 
#include < iostream >
#include < conio.h >
using namespace std;
main()
{
int a,b,c;
a=10,b=10;
c=++a + b;
cout<<"a=10, b=10\n\nThe Sum of ++a + b is "<< c<< "\n\na="<< a<< ", b="<< b;
getch();
}


POST INCREMENT

IT IS REPRESENTED BY (++) AFTER THE VARIABLE NAME.

Syntax-: a++

It increment the value of operand by 1. This operator first returns the value of operand and then it increments its value.

EXAMPLE-: 
#include < iostream >
#include < conio.h >
using namespace std;
main()
{
int a,b,c;
a=10,b=10;
c=(a++) + b;
cout<<"a=10, b=10\n\nThe Sum of (a++) + b is "<< c<< "\n\na="<< a<< ", b="<< b;
getch();
}


TERNARY OPERATOR

The Ternary Operator is an operator which takes 3 arguments:-

ARGUMENTS FUNCTION
FIRST Comparison Argument i.e. Condition
SECOND Result upon if the Condition is True
THIRD Result upon if the Condition is False


If-else statement and Conditional Operator(condition? true: false;) are some types of Ternary Operators.

EXAMPLE-: 
#include < iostream >
#include < conio.h >
using namespace std;
main()
{
int n,n2;
cout<<"THIS IS A PROGRAM TO CHECK THAT THE NUMBERS ENTERED BY USER IS EVEN OR ODD\n\nEnter the Number ";
cin>>n;
//conditional operator
n%2==0?cout<<"EVEN":cout<<"ODD";

cout<<"\n\nEnter Second Number ";
cin>>n2;

//if-else statement
if (n2%2==0)
{
	cout<<"EVEN";
}
else
{
	cout<<"ODD";
}

getch();
}
